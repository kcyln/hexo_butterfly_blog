使用 `hexo_theme_butterfly` 主题搭建的个人博客 

使用 GitHub仓库作为图床，利用PicGo上传

**本地测试**
```
hexo clean
hexo g  
hexo s
```

**部署方式**
```
sh deploy.sh
```

**版本依赖**
- hexo_theme_butterfly  v4.10.0

- node 20.9.0
- npm 10.2.3
- cnpm@9.2.0 npm@9.9.0 node@20.9.0
- hexo 6.3.0
